# Sales Performance & Profitability Analysis (Power BI)

## 📊 Project Overview
This project provides a comprehensive analysis of sales performance, profitability, and COGS for Company. The goal is to identify high-margin products, monitor regional sales trends, and provide actionable insights for revenue optimization.

##  Key Features 
- **Sales Analysis:** Tracked total revenue across different time periods and regions.
- **Profitability Tracking:** Monitored gross profit and profit margins to identify top-performing segments.
- **COGS Management:** Analyzed the relationship between sales volume and production costs.
- **Interactive Slicers:** Dynamic filtering by Year, Region, and Product Category.

## 🛠️ Tools Used
- **Power BI Desktop:** Data modeling, DAX measures, and dashboard creation.
- **Power Query:** Data cleaning (removing nulls, fixing data types, and merging tables).


## 📈 Key Insights
- **Revenue Drivers:** Germany and france contributed 35% of total sales.
- **Profit Margins:** Mexico region has the lowest profit margin due to high logistics COGS].
- **Seasonal Trends:** Sales peaked significantly in USA, suggesting a need for increased inventory.
